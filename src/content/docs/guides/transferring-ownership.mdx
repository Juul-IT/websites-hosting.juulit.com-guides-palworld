## How To Transfer Ownership of a Palworld Server to Another Player

To transfer ownership of your Palworld server to another player, follow these steps:

1. **Access Server Settings:**
   - Locate your Palworld server's configuration files. These are typically named something like `Game.ini` or `GameUserSettings.ini`.

2. **Find Current Owner ID:**
   - Obtain the Steam64 ID of the current server owner. You can find this ID using tools like SteamID.io.

3. **Find New Owner ID:**
   - Obtain the Steam64 ID of the player you want to transfer ownership to.

4. **Edit Ownership Settings:**
   - Open the configuration file and look for a setting that defines the server owner. This setting may be named something like `ServerOwner`.
   - Replace the current owner's Steam64 ID with the Steam64 ID of the new owner.

5. **Save Changes:**
   - Save the configuration file with the new owner's Steam64 ID.

6. **Restart the Server:**
   - Restart your Palworld server for the changes to take effect.

7. **Verify Ownership Transfer:**
   - Join the server in Palworld and verify that the new owner has ownership rights and can manage the server.

By following these steps, you can transfer ownership of your Palworld server to another player.

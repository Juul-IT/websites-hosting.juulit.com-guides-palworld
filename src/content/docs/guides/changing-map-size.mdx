## How To Change the Map Size on a Palworld Server

To change the map size on your Palworld server, follow these steps:

1. **Locate the Map Size Setting:**
   - Access your Palworld server's configuration files. Look for a file named something like `Game.ini` or `GameUserSettings.ini`.

2. **Adjust the Map Size Setting:**
   - Look for a setting that determines the map size. It may be named something like `MapSize` or `MapDimensions`.
   - Change the value to set the desired map size. For example, to increase the map size, you might change the value to `MapSize=20000`.

3. **Save Changes:**
   - Save the configuration file and close the text editor.

4. **Restart the Server:**
   - Restart your Palworld server for the changes to take effect.

5. **Verify the Map Size:**
   - Join the server in Palworld and explore the map to ensure that the size has been adjusted according to your settings.

By following these steps, you can adjust the map size for your Palworld server to create a larger or smaller playable area.
